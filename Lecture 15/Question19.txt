Given:
public class Test {
	public static void main(String[] args) {
		Boolean test = new Boolean(true);
		Integer x = 343;
		Integer y = new Test().go(test, x);
		System.out.println(y);
	}

	int go(Boolean b, int i) {
		if (b)
			return (i / 7);
		return (i / 49);
	}
}
What is the result ?
7
49
343
Compilation fails.
An exception is thrown at runtime.

Ans :-

49
